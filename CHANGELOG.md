# Changelog

## 0.1.0 - 2026-06-21

First explicit version for the Agent Constitution governance framework.

### Added

- Governed-decision doctrine covering decision boundaries, trigger levels, policy outcomes, challenge flow, and compact decision records.
- Decision note recording the governed-decision boundary change.
- README project metadata with version, status, release notes, repository, and keywords.
- Version metadata in `VERSION`.

### Updated

- Adoption, decision-note, handoff, and PR checklist templates now apply the governed-decision model.
- Authority, evidence, destructive-action, provenance, self-update, glossary, and adoption docs now reference or support the governed-decision doctrine.

### Unchanged

- This remains a documentation and template repository, not an application or policy engine.
- No package metadata, CI, automation, or generated artifacts are introduced.
